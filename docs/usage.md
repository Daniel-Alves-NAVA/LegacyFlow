# Uso

Este pacote foi pensado para ser usado como conjunto de guidelines corporativas para GitHub Copilot.

## Fluxo sugerido

1. Selecione uma classe, método, controller, listener ou job no IDE.
2. Use o prompt de geração de documentação.
3. Revise a saída com a skill de revisão.
4. Atualize a memória local quando útil.
5. Faça backup da memória local ao trocar de máquina.
