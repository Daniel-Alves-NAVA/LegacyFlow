# /review-legacy-doc

Use este comando para revisar documentação já gerada e apontar problemas de rastreabilidade, evidência e qualidade.

## Exemplo

`/review-legacy-doc`

## O que o Copilot deve fazer

- validar Mermaid;
- verificar se há evidências;
- identificar alucinações ou inferências sem suporte;
- sugerir melhorias.
