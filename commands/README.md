# Slash Commands for Copilot

Os comandos abaixo foram organizados para facilitar o uso do kit por meio de prompts curtos no Copilot.

## Comandos disponíveis

- `/generate-legacy-flow` — gera documentação técnica e de negócio para um fluxo legado.
- `/review-legacy-doc` — revisa a documentação gerada e aponta problemas.
- `/estimate-context-cost` — estima o custo aproximado de contexto e tokens.
- `/update-local-memory` — atualiza a memória local do projeto.
- `/backup-local-memory` — cria um backup da memória local.

## Como usar

1. Abra o chat do Copilot no editor.
2. Digite um comando começando com `/`.
3. Selecione o código relevante no IDE para que o Copilot use contexto local.
4. Revise a resposta antes de commitar.
