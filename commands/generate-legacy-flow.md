# /generate-legacy-flow

Use este comando para pedir ao Copilot que documente um fluxo legado a partir do código aberto no editor.

## Exemplo

`/generate-legacy-flow`

## O que o Copilot deve fazer

- identificar o entrypoint;
- analisar arquivos relevantes;
- gerar Mermaid técnico e de negócio;
- listar evidências;
- declarar pontos não identificados;
- usar o template de documentação.
